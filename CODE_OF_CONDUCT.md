# Code of Conduct

PompoEngine is intended to be a practical, welcoming open-source project for
visual novel engine development. Everyone participating in the project is
expected to keep discussion focused, respectful, and useful.

## Expected Behavior

- Use clear, constructive language when reporting bugs, reviewing code, or
  discussing design tradeoffs.
- Respect different experience levels and backgrounds.
- Give technical feedback about code, documentation, tests, and project
  decisions without personal attacks.
- Keep issue and pull request discussions on topic.
- Accept maintainer decisions about scope, release readiness, and project
  direction after the reasoning has been shared.

## Unacceptable Behavior

- Harassment, insults, threats, or discriminatory language.
- Sexualized comments or imagery in project spaces.
- Publishing private contact, security, or personal information without
  permission.
- Repeated off-topic disruption after a maintainer asks for the discussion to
  return to the project.
- Abuse of the issue tracker, pull requests, or release channels.

## Reporting

If you see behavior that violates this code, contact the repository maintainers
through the private channels listed by the project owner or by opening a private
security/advisory report when sensitive information is involved.

Maintainers may remove comments, close discussions, restrict participation, or
block accounts when needed to protect the project community.

## Scope

This code applies to project-hosted spaces including issues, pull requests,
discussions, release comments, documentation contributions, and official
community channels associated with PompoEngine.
